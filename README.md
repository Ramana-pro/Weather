# Weather Forecast App


A simple responsive Weather Forecast single-page app built with plain HTML, CSS and JavaScript. It uses the OpenWeatherMap API to fetch current weather.


## Features


- Search weather by city name
- Use device geolocation to get local weather
- Toggle units between °C and °F
- Shows temperature, description, humidity, wind and pressure
- Responsive layout for mobile and desktop


## Live demo
You can run the app locally (no build required). See **Run locally** below.


## Setup
1. Create a free account at [OpenWeatherMap](https://openweathermap.org/) and get an API key.
2. Replace `YOUR_API_KEY_HERE` in `script.js` with your API key.


## Run locally
1. Clone the repo or copy files to a folder.
2. Open `index.html` in your browser.


> If you want to run from a local server (recommended for some browsers):
>
> ```bash
> # using Python 3
> python -m http.server 5500
> # then visit http://localhost:5500
> ```
